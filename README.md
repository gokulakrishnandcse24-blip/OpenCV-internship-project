# OpenCV-internship-project
This project was developed after my first internship at NSIC as part of my learning journey in Computer Vision using OpenCV. It demonstrates an air drawing system using hand gesture recognition, built with Python, OpenCV, and MediaPipe. 
The project creates a gesture-based air drawing program that allows users to use hand gestures recorded by a webcam to draw on a virtual canvas.

OpenCV is used to render the drawing interface and process the camera feed, while MediaPipe Hand Landmarker is used for real-time hand tracking.

Technologies Employed

Python

OpenCV: drawing canvas, image processing, and video capture
Real-time hand landmark detection with MediaPipe
NumPy: array operations for manipulating canvases

Features

*Drawing with pointer finger 
*Clickable color scheme on the screen
*Erase mode with 3 finger motions
*Option for a clear canvas
*Smooth illustration using landmark smoothing
*Webcam-based real-time hand tracking

I gained a better understanding of real-time computer vision applications, gesture recognition, and hand landmark detection thanks to this project.
